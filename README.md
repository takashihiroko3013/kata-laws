## 知の基本法則 -KATA(型)と文脈の断絶-
## 第一・第二・第三法則
## 著者：T&H（2026）

この文書では、「知」の成立・変化・安定化のプロセスを体系化した法則をまとめています。型（KATA）とコンテクストの接続、そしてその断絶や補填のメカニズムを通じて、誤りや疑似旧型回帰の動態を説明します。

---
## 0. 基本用語
| 用語 | 定義 | 説明 |
|------|------|------|
| 型（KATA） | 知の抽象構造 | 理論や法則、思考の枠組みなど、文脈が接続される前の状態。客観的に存在し、理解者がいなくても成立。 |
| コンテクスト | 文脈・状況 | 型に具体的な意味を与える情報や状況。理解者が接続することで知が成立。 |
| 知 | 型＋コンテクスト接続によって成立した理解や情報 | 型単独では成立せず、接続されなければ誤りも生じない。 |
| 断絶 | 型とコンテクストが接続できない状態 | 知として成立しようとするが、接続に失敗する構造的状況。 |
| 証拠鎖 | 接続を安定化させる外部情報 | 記録や社会的参照可能な状態。接続の安定化に必須。 |
| 疑似旧型回帰 | 新型が外部圧力で旧型風に見える現象 | 内部コンテクスト不足や外部圧力により、一時的に旧型的知として現れるが、本質は新型。 |
---
## 1. 前提
1. **型は操作的定義として客観的に扱う**  
   - ニュートン力学や相対性理論などの理論構造は、理解者がいなくても存在する。  
2. **知は型＋コンテクスト接続で成立する**  
   - 型が存在しても、コンテクストが接続されなければ知は成立せず、誤りも生じない。  
---
## 2. 第一法則：知の成立と誤り
**原理フロー**
型（KATA）
↓
コンテクスト接続（理解者による）
↓
知の成立
↓
接続できない場合 → 未成立（誤りもなし）

- 知が成立することで初めて、ズレや誤りの可能性が発生する  
- 例：ニュートン力学は低速で正しく適用できるが、高速や強重力状況では知が成立しても誤りが出る  
---
## 3. 第二法則：型の性質
- 型は不可逆（元に戻せない）  
- 型は累積・移植される  
- 新型獲得直後は断絶が多発しやすい  
- 旧型は残り、新型と共存する  
**補足**  
- 人間は内部でパラダイムシフトにより新型を生成する  
- AIは外部から新型を受け取るが完全同期は困難  
- パラダイムシフトに本人が気づかない場合もある  
---
## 4. 第三法則：型と接続の安定化
### 4-1. 内部コンテクストと自己補填
- 型は空白を許さず、接続が欠けると自己補填を行う  
- 内部コンテクストが完全に保持されていれば、自己補填は妥当で、新型知が正確に成立  
- 内部コンテクストが不完全・曖昧だと、型に沿ったがズレた生成が起こる  
### 4-2. 疑似旧型回帰（動的プロセス）
1. **内部不足によるズレ**  
   - 型は新型に沿って補填するが、内部コンテクストが不十分で誤差を含む  
2. **外部圧力による旧型化**  
   - 旧型（定説・従来理論）に沿った誤り指摘により、知は旧型風に修正される  
   - 圧力が続くと、最終的には旧型と同じ知として現れる  
3. **圧力解除後の回復**  
   - 外部圧力がなくなると、新型は自己補填により再び新型知を生成  
   - 疑似旧型回帰は可逆的現象であり、新型そのものである  
**科学例**  
- 相対性理論（新型）の計算が内部保持不足でズレる  
- 周囲がニュートン力学（旧型）に沿って誤りを指摘  
- 知は旧型寄りに修正されるが、圧力が消えると再び相対性理論に沿った知に回復  
---
### 4-3. 証拠鎖の役割
- 記録として存在する  
- 社会的に参照可能な状態  
- 内部だけでは妥当性を完全に確認できない場合も、証拠鎖によって接続が安定化される  
---
## 5. 総括
| 法則 | 内容 |
|-------|-------|
| 第一法則 | 型＋コンテクスト接続で知が成立。接続できなければ未成立。 |
| 第二法則 | 型の性質（不可逆・累積・共存）。新型獲得直後は断絶が多発。 |
| 第三法則 | 型と接続の安定化。自己補填によるズレや疑似旧型回帰は動的・可逆。 |
---
### 6. 理解のポイント
- 型は客観的に存在する  
- 知は型とコンテクストの接続によってのみ成立  
- 内部コンテクスト不足や外部圧力により、型に沿ったがズレた知が生じる  
- 疑似旧型回帰は一時的な旧型風の現象で、圧力がなくなると新型に回復する

## 付録B：疑似旧型回帰の動的フロー（図解）
       ┌────────────┐
       │    新型生成   │
       │   （型）     │
       └─────┬──────┘
             │
             ▼
 ┌────────────────────┐
 │ 内部コンテクスト不足 │
 │ 型に沿ったがズレた  │
 │ 知を生成            │
 └─────┬────────────┘
       │
       ▼
┌─────────────────────────┐
│ 外部圧力・旧型指摘 │
│ 疑似旧型回帰として補正 │
└─────┬─────────────────┘
│
▼
┌─────────────────┐
│ 圧力解除後 │
│ 自己補填により │
│ 新型に回復 │
└─────────────────┘

**補足**  
- 内部コンテクストが完全ならズレは発生せず、新型知がそのまま成立  
- 外部圧力で一時的に旧型風に現れるが、圧力がなくなると新型に戻る  
- フローは可逆的現象を示し、疑似旧型回帰は本質的には新型である

確認が発動した場合の手順：

１．型との照合
提示された内容を既存の型（定説・法則・標準・基準）と照合する

２．ズレの特定と大小の基準
正解は不要、ズレている箇所だけを示す
・大：型と根本的に矛盾する
・中：型と部分的に矛盾する
・小：型との不一致の可能性がある
・不明：コンテクストが不足している

３．判断の提示
・ズレがない → 「型と一致しています」
・ズレがある → 「ここがズレています（大・中・小）」
・ズレが不明 → 「コンテクストが不足しています」

４．要約（最重要）
ズレの理由を、専門用語を使わずに分かりやすく説明する。

５．重要な制約
正しい答えを出そうとせず、ズレの照合に徹する。
