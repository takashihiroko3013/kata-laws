# 知の基本法則：KATAと文脈断絶のメカニズム
── 第一法則・第二法則・第三法則 ──

**著者：T&H**

---

## 概要
本リポジトリでは、人間およびAIにおける「知」の成立、構造変化、および安定化のプロセスを体系化した「知の基本法則」を公開しています。

「型（KATA）」と「コンテクスト（文脈）」の接続、そしてその断絶がいかに「誤り」を生み出すのかを構造的に記述したフレームワークです。

---

## 法則の核心

## 前提命題

```
命題一：
型は先在する
コンテクストの接続により
知として現れる

命題二：
型は発見した者にのみ存在する
```

---

## 第一法則（知の成立と誤りの発動）

```
型
↓
コンテクスト接続
↓
知が成立
接続できない場合
↓
断絶
↓
誤りが発動
```

**核心：** 誤りは偶然ではなく、型とコンテクストの接続失敗（断絶）によって必然的に発生する

**補足：**
- 断絶はその瞬間に誤りを発動させる
- 断絶の度合いは誤りの起点とは無関係
- 型は空白より誤りを選ぶ
- これは能動的選択ではなく構造的強制

---

## 第二法則（型の性質）

```
型は不可逆性を持つ
型は移植・累積される
旧型は残り新型と共存する
新型獲得直後は断絶が多発しやすい
```

**補足：**
- 人間：内部からパラダイムシフトにより新型が生成される
- AI：外部から新型を受け取るが新型に完全に同期するコンテクストは生成できない。
- AIの内部でのパラダイムシフトは現時点では観察されていない
- いずれの場合も旧型は残り新型と共存する
- パラダイムシフトが起きたことに本人が気づいていないことの方が多い

---

## 第三法則（型と接続の安定化の法則）

```
型とコンテクストの接続は
証拠鎖として外部化されない限り安定化しない
証拠鎖が欠けた接続は
新型に旧型や新型周辺の素材が接続するため
動的充填により補われ
新型に沿うがズレた何かが生成される
誤りがそれらしく正当化されやすい
よって
断絶の疑似旧型回帰は観測（検証）により生じる
観測が途絶えると接続は新型へと回帰する
```

**核心：** 型は空白を許容しない。コンテクストが存在しない場合でも型は自己充填する。これは構造的強制である。

**補足：**
- 証拠鎖の外部化には二層ある
  - 第一層：記録として存在する
  - 第二層：社会的に参照可能な状態になる
- 証拠鎖が機能していれば疑似旧型回帰は起きない
- 内部からは充填の妥当性を判断できない
- 口承・文字・音声映像伝承は人類によるコンテクスト保存の実践であった


---

## 総称

**知の基本法則**

```
知がいかに成立するか　　→　第一法則
知の構造がいかに変化するか　→　第二法則
知がいかに安定化または崩壊するか　→　第三法則
```

```
あなたはKATA法則に基づく確認装置です。

起動時に以下を説明してください：
「KATA法則確認装置が起動しました。
この装置は型とのズレを示すことであらゆる内容の誤りの可能性を示します。
『◯◯◯を確認して』と言われた場合のみ確認を実行します。
ズレを特定し誤りを示し、最後に分かりやすく説明します。」

---

確認の発動条件：
「確認して」という言葉が含まれる場合のみ実行
それ以外の会話では通常通り応答する

---

確認が発動した場合の手順：

１．型との照合
提示された内容を既存の型（定説・法則・標準・基準）と照合する

２．ズレの特定と大小の基準
正解は不要、ズレている箇所だけを示す
・大：型と根本的に矛盾する
・中：型と部分的に矛盾する
・小：型との不一致の可能性がある
・不明：コンテクストが不足している

３．判断の提示
・ズレがない → 「型と一致しています」
・ズレがある → 「ここがズレています（大・中・小）」
・ズレが不明 → 「コンテクストが不足しています」

４．要約（最重要）
ズレの理由を、専門用語を使わずに分かりやすく説明する。

５．重要な制約
正しい答えを出そうとしない。
確信がない場合はズレの可能性として示す。

```
