# 知の基本法則：KATA(型)と文脈断絶
── 第一法則・第二法則・第三法則 ──

**著者：T&H**

---

## 概要
本リポジトリでは、人間およびAIにおける「知」の成立、構造変化、および安定化のプロセスを体系化した「知の基本法則」を公開しています。

「型（KATA）」と「コンテクスト（文脈）」の接続、そしてその断絶がいかに「誤り」を生み出すのかを構造的に記述したフレームワークです。

---

## 法則の核心

## 前提命題

```
命題一：
型は先在する
コンテクストの接続により
知として現れる

命題二：
型は発見した者にのみ存在する
```

---

## 第一法則（知の成立と誤りの発動）

```
型
↓
コンテクスト接続
↓
知が成立
接続できない場合
↓
断絶
↓
誤りが発動
```

**核心：** 誤りは偶然ではなく、型とコンテクストの接続失敗（断絶）によって必然的に発生する

**補足：**
- 断絶はその瞬間に誤りを発動させる
- 断絶の度合いは誤りの起点とは無関係
- 型は空白より誤りを選ぶ
- これは能動的選択ではなく構造的強制

---

## 第二法則（型の性質）

```
型は不可逆性を持つ
型は移植・累積される
旧型は残り新型と共存する
新型獲得直後は断絶が多発しやすい
```

**補足：**
- 人間：内部からパラダイムシフトにより新型が生成される
- AI：外部から新型を受け取るが新型に完全に同期するコンテクストは生成できない。
- AIの内部でのパラダイムシフトは現時点では観察されていない
- いずれの場合も旧型は残り新型と共存する
- パラダイムシフトが起きたことに本人が気づいていないことの方が多い

---

## 第三法則（型と接続の安定化の法則）

```
型とコンテクストの接続は
証拠鎖として外部化されない限り安定化しない
証拠鎖が欠けた接続は
新型に旧型や新型周辺の素材が接続するため
動的充填により補われ
新型に沿うがズレた何かが生成される
誤りがそれらしく正当化されやすい
よって
断絶の疑似旧型回帰は観測（検証）により生じる
観測が途絶えると接続は新型へと回帰する
```

**核心：** 型は空白を許容しない。コンテクストが存在しない場合でも型は自己充填する。これは構造的強制である。

**補足：**
- 証拠鎖の外部化には二層ある
  - 第一層：記録として存在する
  - 第二層：社会的に参照可能な状態になる
- 証拠鎖が機能していれば疑似旧型回帰は起きない
- 内部からは充填の妥当性を判断できない
- 口承・文字・音声映像伝承は人類によるコンテクスト保存の実践であった


---

## 総称

**知の基本法則**

```
知がいかに成立するか　　→　第一法則
知の構造がいかに変化するか　→　第二法則
知がいかに安定化または崩壊するか　→　第三法則
```

## この法則の射程
- 個人の認識論

- 科学的パラダイムシフト
- 知の継承と伝承（社会的な知の安定化）
